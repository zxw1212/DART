## 一、核心训练模块 (`mld/`)

### 训练脚本

#### `mld/train_mvae.py` ⭐
**作用**: 训练运动原语VAE（Motion Primitive VAE）

**功能**:
- 将运动序列编码为紧凑的潜在向量
- 学习运动的低维表示
- 支持SMPLX和SMPLH身体模型

**关键参数**:
```python
--exp_name: 实验名称
--data_args.dataset: 数据集类型 ('mp_seq_v2', 'hml3d')
--data_args.data_dir: 数据目录
--data_args.cfg_path: 运动原语配置文件
--model_args.latent_dim: 潜在向量维度 (通常256)
--train_args.batch_size: 批次大小 (128)
--train_args.stage1_steps: 训练步数 (100000)
```

**输出**: `./mvae/{exp_name}/checkpoint_{step}.pt`

---

#### `mld/train_mld.py` ⭐
**作用**: 训练潜在扩散模型（Motion Latent Diffusion）

**功能**:
- 在VAE潜在空间训练扩散模型
- 支持文本条件生成
- 实现分类器自由引导

**关键参数**:
```python
--exp_name: 实验名称
--denoiser_args.mvae_path: 预训练VAE路径
--train_args.batch_size: 批次大小 (1024)
--train_args.use_amp: 是否使用混合精度训练
--train_args.stage1_steps: 训练步数 (100000)
```

**输出**: `./mld_denoiser/{exp_name}/checkpoint_{step}.pt`

---

### 推理脚本

#### `mld/rollout_mld.py` ⭐
**作用**: 自回归运动生成（批量模式）

**功能**:
- 根据文本提示生成长序列运动
- 支持运动组合（多个动作拼接）
- 导出SMPLX参数和NPZ文件

**使用示例**:
```bash
python -m mld.rollout_mld \
    --checkpoint './mld_denoiser/xxx/checkpoint_300000.pt' \
    --text_prompt "walk forward*5,turn left*3" \
    --guidance_param 5.0
```

**输出**: PKL和NPZ文件

---

#### `mld/rollout_demo.py` ⭐
**作用**: 交互式实时运动生成

**功能**:
- 实时3D可视化（pyrender）
- 命令行文本输入
- 流式SMPLX渲染

**使用**:
```bash
source ./demos/run_demo.sh
```

**特点**:
- 实时响应（~20 FPS）
- 支持动态文本输入
- 自动相机跟随

---

### 优化脚本

#### `mld/optim_mld.py` ⭐
**作用**: 潜在噪声优化（满足约束）

**功能**:
- 运动插值（inbetweening）
- 轨迹控制
- 约束满足优化

**约束类型**:
- 地面接触约束
- 运动平滑约束（jerk）
- 目标位置约束
- 脚部滑动约束

**使用场景**:
```bash
# 运动插值
source ./demos/inbetween_babel.sh

# 轨迹控制
source ./demos/traj.sh
```

---

#### `mld/optim_scene_mld.py` ⭐
**作用**: 场景交互优化

**功能**:
- 人-场景交互生成
- 碰撞检测和避免
- 接触约束

**额外约束**:
- SDF碰撞约束
- 接触点约束
- 场景几何约束

**使用**:
```bash
source ./demos/scene.sh
```

---

### 模型架构 (`mld/models/`)

#### `mld/models/architectures/mld_vae.py`
**定义**: `MldVae` 类

**架构**:
```python
class MldVae:
    encoder: TransformerEncoder  # 编码器
    decoder: TransformerDecoder  # 解码器
    mu_layer: Linear            # 均值层
    logvar_layer: Linear        # 方差层
```

**输入/输出**:
- 输入: (batch, seq_len, 263) - SMPLX参数
- 输出: (batch, 1, latent_dim) - 潜在向量

---

#### `mld/models/architectures/mld_denoiser.py`
**定义**: `MldDenoiser` 类

**架构**:
```python
class MldDenoiser:
    time_embed: TimestepEmbedding  # 时间步嵌入
    text_encoder: CLIPTextEncoder  # 文本编码器
    denoiser: Transformer/MLP      # 去噪网络
```

**输入/输出**:
- 输入: 噪声潜在向量 + 时间步 + 文本
- 输出: 预测的噪声

## 二、强化学习控制 (`control/`)

#### `control/train_reach_location_mld.py` ⭐
**作用**: 训练目标到达控制策略

**算法**: PPO (Proximal Policy Optimization)

**功能**:
- 训练策略网络选择潜在动作
- 实现目标导向的运动控制
- 支持多种运动技能（走、跑、跳）

**关键参数**:
```python
--denoiser_checkpoint: 扩散模型检查点
--env_args.num_envs: 并行环境数 (256)
--env_args.texts: 运动技能列表 ['walk', 'run', 'hop on left leg']
--learning_rate: 学习率 (3e-4)
--total_timesteps: 总训练步数 (200M)
```

**输出**: `./policy_train/{exp_name}/`

---

#### `control/env/env_reach_location_mld.py`
**定义**: `EnvReachLocationMLD` 环境类

**状态空间**:
- 目标方向和距离
- 文本嵌入
- 运动历史

**动作空间**:
- 潜在噪声向量

**奖励函数**:
- 目标到达奖励
- 地面接触惩罚
- 脚部滑动惩罚
- 姿态惩罚

---

#### `control/policy/policy.py`
**定义**: 策略网络类

**架构**:
- `PolicyReachLocationMLP`: MLP策略
- `PolicyReachLocationTransformer`: Transformer策略

**输出**: 高斯分布参数 (mean, log_std)

## 四、可视化工具 (`visualize/`)

**参数**:
- `--add_floor`: 添加地面
- `--translate_body`: 身体平移

## 十五、工作流程文件依赖

### 训练流程

```
1. 数据准备
   data_scripts/extract_dataset.py
   ↓
2. 训练VAE
   mld/train_mvae.py
   ↓
3. 训练扩散模型
   mld/train_mld.py
   ↓
4. 训练策略（可选）
   control/train_reach_location_mld.py
```

### 推理流程

```
1. 加载模型
   mld/rollout_mld.py 或 mld/rollout_demo.py
   ↓
2. 文本编码
   utils/text_encoders.py
   ↓
3. 扩散采样
   diffusion/gaussian_diffusion.py
   ↓
4. VAE解码
   model/mld_vae.py
   ↓
5. 可视化
   visualize/vis_seq.py
```

## 十六、重要配置参数说明

### VAE训练参数
```python
--model_args.latent_dim: 潜在维度 (128/256)
--model_args.num_layers: Transformer层数 (7)
--train_args.weight_kl: KL散度权重 (1e-6)
--train_args.weight_smpl_joints_rec: 关节重建权重 (10.0)
```

### 扩散模型参数
```python
--denoiser_args.diffusion_steps: 扩散步数 (1000)
--denoiser_args.noise_schedule: 噪声调度 ('linear'/'cosine')
--train_args.weight_latent_rec: 潜在重建权重 (1.0)
```

### 推理参数
```python
--guidance_param: 引导强度 (5.0)
--respacing: DDIM步数 ('ddim10')
--seed: 随机种子
```

### 优化参数
```python
--optim_lr: 优化学习率 (0.05)
--optim_steps: 优化步数 (100)
--weight_floor: 地面约束权重 (1.0)
--weight_jerk: 平滑约束权重 (0.1)
--weight_goal: 目标约束权重 (1.0)
```
